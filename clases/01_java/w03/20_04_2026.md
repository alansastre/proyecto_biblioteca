## Arquitectura Spring Boot MVC

```
Usuario (navegador)
     │
     ▼
Controller ── recibe la petición HTTP (localhost:8080/restaurants)
     │
     ▼
Repository ── consulta entidades en la base de datos (findAll, findById...)
     │
     ▼
HTML (Thymeleaf) ── muestra los datos en una tabla bonita
```

El **controlador** es la pieza que conecta lo que el usuario pide en el navegador con los datos de la base de datos.

## Crear RestaurantController

Nuevo archivo: `src/main/java/com/restaurantes/controller/RestaurantController.java`

```java
package com.restaurantes.controller;

import com.restaurantes.repository.RestaurantRepository;
import org.springframework.stereotype.Controller;
import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.GetMapping;

@Controller
public class RestaurantController {

    private final RestaurantRepository restaurantRepository;

    public RestaurantController(RestaurantRepository restaurantRepository) {
        this.restaurantRepository = restaurantRepository;
    }

    @GetMapping("/restaurants")
    public String findAll(Model model) {
        model.addAttribute("restaurants", restaurantRepository.findAll());
        return "restaurant-list";
    }
}
```

Línea a línea:
* `@Controller` — le dice a Spring que esta clase recibe peticiones HTTP del navegador
* Constructor con `RestaurantRepository` — Spring inyecta el repositorio automáticamente (igual que con `context.getBean`)
* `@GetMapping("/restaurants")` — este método se ejecuta cuando alguien navega a `localhost:8080/restaurants`
* `Model model` — es una caja donde metemos datos para enviar al HTML
* `model.addAttribute("restaurants", ...)` — guarda la lista con el nombre `"restaurants"` para usarlo en el HTML
* `return "restaurant-list"` — Spring busca el archivo `templates/restaurant-list.html` y lo muestra

## Crear restaurant-list.html

Nuevo archivo: `src/main/resources/templates/restaurant-list.html`

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Restaurantes</title>
</head>
<body>

    <h1>Lista de restaurantes</h1>

    <table border="1">
        <thead>
            <tr>
                <th>ID</th>
                <th>Nombre</th>
                <th>Precio medio</th>
                <th>Tipo de comida</th>
                <th>Fecha apertura</th>
            </tr>
        </thead>
        <tbody>
            <tr th:each="restaurant : ${restaurants}">
                <td th:text="${restaurant.id}"></td>
                <td th:text="${restaurant.name}"></td>
                <td th:text="${restaurant.averagePrice}"></td>
                <td th:text="${restaurant.foodType}"></td>
                <td th:text="${restaurant.startDate}"></td>
            </tr>
        </tbody>
    </table>

</body>
</html>
```

Thymeleaf:
* `th:each="restaurant : ${restaurants}"` — es un bucle for: recorre la lista `restaurants` que metimos en el `model`
  * Equivale al `for (Restaurant restaurant : restaurants)` que ya conocemos de Java
* `th:text="${restaurant.name}"` — muestra el valor de `getName()` del restaurante
* `${restaurants}` — accede al dato que guardamos con `model.addAttribute("restaurants", ...)`

## DataInitializer: limpiar el main

El `main()` tiene muchas líneas creando datos de prueba. Lo movemos a una clase aparte que se ejecuta automáticamente al arrancar:

Nuevo archivo: `src/main/java/com/restaurantes/config/DataInitializer.java`

```java
package com.restaurantes.config;

import com.restaurantes.model.*;
import com.restaurantes.repository.*;
import org.springframework.boot.CommandLineRunner;
import org.springframework.stereotype.Component;

import java.time.LocalDateTime;
import java.util.List;

@Component
public class DataInitializer implements CommandLineRunner {

    private final RestaurantRepository restaurantRepository;
    private final EmployeeRepository employeeRepository;
    private final DishRepository dishRepository;
    private final OrderRepository orderRepository;
    private final OrderLineRepository orderLineRepository;

    public DataInitializer(RestaurantRepository restaurantRepository,
                           EmployeeRepository employeeRepository,
                           DishRepository dishRepository,
                           OrderRepository orderRepository,
                           OrderLineRepository orderLineRepository) {
        this.restaurantRepository = restaurantRepository;
        this.employeeRepository = employeeRepository;
        this.dishRepository = dishRepository;
        this.orderRepository = orderRepository;
        this.orderLineRepository = orderLineRepository;
    }

    @Override
    public void run(String... args) {

        // Restaurantes
        Restaurant taberna = new Restaurant("La Taberna", 15.0, 4);
        taberna.setFoodType(FoodType.SPANISH);
        Restaurant dominos = new Restaurant("Dominos Pizza", 12.0, 6);
        restaurantRepository.saveAll(List.of(taberna, dominos));

        // Empleados
        Employee emp1 = new Employee("Juanito", "García", 25, "12345678A");
        emp1.setRestaurant(taberna);
        Employee emp2 = new Employee("Patricia", "López", 35, "87654321B");
        emp2.setRestaurant(dominos);
        employeeRepository.saveAll(List.of(emp1, emp2));

        // Platos
        Dish ensalada = new Dish(null, "Ensalada", "Mixta con tomate", 5.0, DishType.STARTER, taberna);
        Dish lentejas = new Dish(null, "Lentejas", "Con chorizo", 8.0, DishType.MAIN, taberna);
        Dish tarta = new Dish(null, "Tarta de queso", null, 7.50, DishType.DESSERT, taberna);
        dishRepository.saveAll(List.of(ensalada, lentejas, tarta));

        // Pedido
        Order pedido1 = new Order();
        pedido1.setDate(LocalDateTime.now());
        pedido1.setTableNumber(5);
        pedido1.setNumPeople(3);
        pedido1.setStatus(OrderStatus.PENDING);
        pedido1.setRestaurant(taberna);
        orderRepository.save(pedido1);

        // Líneas de pedido
        OrderLine linea1 = new OrderLine(2, ensalada, pedido1);
        OrderLine linea2 = new OrderLine(1, lentejas, pedido1);
        OrderLine linea3 = new OrderLine(3, tarta, pedido1);
        orderLineRepository.saveAll(List.of(linea1, linea2, linea3));

        // Calcular total
        Double total = orderLineRepository.calculateTotalPrice(pedido1.getId());
        pedido1.setTotalPrice(total);
        orderRepository.save(pedido1);
    }
}
```

* `@Component` — Spring crea esta clase automáticamente al arrancar
* `CommandLineRunner` — el método `run()` se ejecuta al iniciar la app
* Así el `main()` queda limpio: solo `SpringApplication.run(...)`

Después de crear el DataInitializer, limpiar el main:

```java
@SpringBootApplication
public class RestaurantesJavaApplication {
    public static void main(String[] args) {
        SpringApplication.run(RestaurantesJavaApplication.class, args);
    }
}
```
