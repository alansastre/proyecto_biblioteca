## Asociación @ManyToOne
* Idea: un empleado trabaja en un restaurante. **Muchos** empleados pueden trabajar en **un** restaurante.
* En base de datos: clave foránea `restaurant_id` en la tabla `employees`
* Equivalente SQL:
  ```sql
  CREATE TABLE employees (
      id BIGINT PRIMARY KEY,
      first_name VARCHAR(255),
      restaurant_id BIGINT,
      FOREIGN KEY (restaurant_id) REFERENCES restaurantes(id)
  );
  ```
