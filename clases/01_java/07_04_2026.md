Archivos java (Arquitetura Spring boot)

* config/ Configuraciones globales, seguridad, inicialización datos,

* controller/ Clases java que reciben peticiones HTTP permite que el usuario pueda navegar entre pantallas
  * Son clases con métodos que tienen return con el nombre del archivo html al que queremos ir

* model/
  * Clases con la información de nuestra aplicación. Depende de la temática o negocio.
  * Suelen tener atributos, constructor, getter y setter

* repository/ 
  * Interfaces para operaciones con base de datos, por ejemplo para guardar y recuperar datos
  * Los generamos automáticamente con el plugin JPA Buddy

* service/ y util/ (opcional)
  * lógica de negocio, cálculos, conexiones a sistemas externos
