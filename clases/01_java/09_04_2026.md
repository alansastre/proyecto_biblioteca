

* [ok] Repositorio
  * Archivos java que permiten hacer operaciones en la base de datos
  * Son interfaces de java, solo declaran métodos, no tienen código
  * findById (SELECT * from ... where id = 1;)
  * findAll (SELECT * from ...;)
  * save (INSERT o UPDATE)
  * delete (DELETE FROM)
  * Se crean de dos formas:
    * Opción 1: usando el plugin JPA Buddy, te colocas en una entidad y pulsas crear Spring Data Repository
    * Opción 2: crearlo manualmente New Java Class Interface

* [ok] Constructor, Getter, Setter, toString
  * Constructor: método especial para crear objetos de una clase
  * Getter: método para obtener el valor de un atributo de una clase
  * Setter: método para cambiar el valor de un atributo de una clase
  * toString: método para convertir un objeto en texto, útil para imprimir por consola, sirve para imprimir en la terminal los datos del objeto, los atributos con sus valores

* Operaciones con repositorios:
  * [ok] Crear objetos Restaurant
  * [ok] Crear objetos Employee
  * [ok] Guardar objetos Restaurant en la base de datos con restaurantRepository.save(restaurant);
  * [ok] Guardar objetos Employee en la base de datos con employeeRepository.save(employee);
  * [ok] Recuperar todos los restaurantes con restaurantRepository.findAll();
  * [ok] Recuperar todos los empleados con employeeRepository.findAll();
  * [ok] for (Restaurant restaurant : restaurantRepository.findAll())
  * [ok] for (Employee employee : employeeRepository.findAll())


* Clases y objetos:
  * Clase: plantilla o molde para crear objetos, define atributos y métodos
    * public class Restaurant {}
    * @Entity
  * Objeto: instancia de una clase, tiene valores concretos para los atributos definidos en la clase
    * Restaurant restaurant1 = new Restaurant();
  * Ejemplo: clase Restaurant, objeto restaurant1 con name = "Pacos Bar",
  * Cosas habituales dentro de una clase Entity:
    * Atributos o propiedades: variables que definen las características de la clase
    * Constructor: método especial para crear objetos de la clase
    * Getter: método para obtener el valor de un atributo
    * Setter: método para cambiar el valor de un atributo
    * toString: método para convertir un objeto en texto, útil para imprimir por consola
