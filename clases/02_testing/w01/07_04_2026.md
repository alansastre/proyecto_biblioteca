

* Spring Boot arquitectura desarrollo
  * Spring Boot framework aplicaciones web https://spring.io

  * config/
    * seguridad global
    * configuración global
    * Encoder de contraseña PasswordEncoder

  * model/
    * Los datos o información con la que trabaja una aplicación 
    * cada modelo es una clase Java que corresponde a una tabla en base de datos
    * Se generan automáticamente las tablas de base de datos
    * Se puede generar opcionalmente las clases java a partir de tablas

  * repository/
    * Interfaces con operaciones de base de datos
    * SELECT: findAll() buscar todos los libros 
    * INSERT: save()
    * UPDATE: save()
    * DELETE: delete()
    * No tenemos que implementar código SQL, lo hace Spring Data JPA con Hibernate
    * Se puede generar automáticamente con el plugin JPA Buddy

  * controller/
    * Peticiones HTTP entrantes
    * @GetMapping   SELECT
    * @PostMapping    CREATE
    * @PutMapping    UPDATE
    * @PatchMapping   UPDATE
    * @DeleteMapping    DELETE
    * Métodos de HTTP
    * Testing: Postman, OpenAPI Swagger, Spring Test
    * Pueden ser de dos tipos:
      * MVC --> HTML
      * REST --> JSON

  * service (opcional)
    * Lógica de negocio
    * Cálculos
    * Conexiones a otros backend

  * src/main/resources/templates/*.html 
    * Vista, Interfaz de usuario, UI, UI/UX
    * HTML
    * Atributos de Thymeleaf, es un motor de plantillas 
    * Se testea con frameworks de testing de interfaz de usuario como Selenium WebDriver, Selenium IDE, Playwright


* Spring Boot arquitectura testing
  * seguir la misma arquitectura de desarrollo
  * Suites de testing, permiten ejecutar todos los tests o grupos de tests


Pirámide de testing:

* Testing unitario
  * Test pequeños
  * Se prueba una función java
  * rápidos
  * aislados
  * independientes
  * JUnit 6, ya viene preconfigurado en Spring: @Test

* Testing de integración
  * Prueba como integran componentes
  * Cómo se integra controller con service con repository
  * @SpringBootTest

* Testing de interfaz de usuario
  * Selenium
  * Tener la aplicación corriendo
  * test de junit 6 con código de selenium que abre el navegador y navega por la aplicación
