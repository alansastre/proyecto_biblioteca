

* GitHub.com crear cuenta 

* JetBrains IntelliJ IDEA
  * Alternativa: Visual Studio Code

* JDK 25 LTS

* Crear proyecto nuevo: New > Project > Java

* Ver commits

* Instalar Git:
  * https://git-scm.com/install/ Git for Windows/x64 Setup.
  * Libro https://git-scm.com/book/en/v2
  * Configuración
    * git config --global user.name "alansastre"
    * git config --global user.email alan@certidevs.com
    * Comprobar:
      * git config --list --show-origin

* Clonar proyecto biblioteca
  * Opción 1: git clone https://github.com/alansastre/proyecto_biblioteca
  * Opción 2: IntelliJ IDEA > New > Project From Version Control


En Settings plugins:

* Extensión lombok
* Extensión JPA Buddy
* Extensión GitHub Copilot (opcional)

* Settings > Annotations Processor > Enable annotation processing

* CUIDADO: si el plugin maven está deshabilitado no detectará bien el proyecto.


* Instalar GitHub desktop (opcional)
  * Cliente GUI de git, que permite trabajar con repositorios git visualmente, similar a vscode o intellij pero específico para git.
  * https://desktop.github.com/download/

  
* Crear proyecto Spring Boot
  * Dependencias: web, spring data jpa, h2, thymeleaf
